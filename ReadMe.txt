1. После загрузки в папке проекта необходимо создать папку .venv

2. Прописать в командой строке в папке проекта pipenv install
Будет  создана виртуальная среда и подгрузятся все необходимые библиотеки

3. Виртуальную среду надо будет активировать с помощью команды pipenv shell

Далее, скорее всего, среда будет активироваться автоматически
В любом случае можно проверить нахождение в среде с помощью команды pip list или иными способами

4. (Пункт можно пропустить и перейти к 5) В виртуальной среде надо будет установить необходимые модули, если те не подгрузились автоматом.
pip install django
pip install stripe
pip install pillow

5. Вводим в командой строке следующий текст:
python manage.py runserver

Слово python обязательно

На этом этапе в браузере по пути 
127.0.0.1:8000 
запуститься сайт со всем необходимым функционалом для пользователей


6. Если необходимо добавить категории или товары, то нужно перейти по пути 127.0.0.1:8000/admin
Однако для этого не хватит уровня доступа обычного пользователя

В папке проекта в командую строку вводим:
python manage.py createsuperuser

Далее предложат создать имя пользователя
Электронную почту можно пропустить
Далее предложат создать пароль

Данные этого суперпользователя позволят войти в админ панель


Для оплаты можно использовать тестовую карту 
4242 4242 4242 4242 
Остальные значения опциональны
